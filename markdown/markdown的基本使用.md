# markdown的基本使用:
## 一.标题

markdown的标题就是#开头，然后几个#就是几级标题，markdown只支持到6级标题，后面加上空格加上标题描述  
例如: 
```markdown
# 这是标题一
## 这是标题二
### 这是标题三
#### 这是标题四
##### 这是标题五
###### 这是标题六
```
实际效果
# 这是标题一
## 这是标题二
### 这是标题三

#### 这是标题四
##### 这是标题五
###### 这是标题六

## 二.段落
段落没啥语法上的要求，直接写就是了，段落换行
如果a b c d直接在d前面按换行符是不会换行的  
markdown写法：
```
a b c
d
```
实际效果：  
a b c
d  
要换行的话要在段尾加两个空格，然后换行才有效果
markdown写法
```
a b c  
d
```
实际效果:  
a b c  
d

## 五.代码
```markdown

```

```cpp
#include <iostream>
using namespace std;
int main(){
    for(int i = 0; i < 3; i++){
        count << '6';
    }
    return 0;
```
